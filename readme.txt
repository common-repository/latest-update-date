== Description ==

For everyone who feels nostalgic and want to put a 'latest update date' in their footer, this is the plugin.

Read more about this plugin on the [Latest Update Date plugin](http://coenjacobs.net/wordpress/plugins/latest-update-date) page.

== Installation ==

1. Upload the `latest-update-date` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Set the settings for displaying the date in the footer
1. Enjoy!
